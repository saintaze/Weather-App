# Weather-App
An app for checking weather. 

### Features
+ Finds weather automatically based on GeoLocation.
+ Can search weather by giving city name.
+ Shows temperature both in Celsius and Fahrenheit.

##### Built With
+ HTML5
+ CSS3
+ Javascript (Vanilla)
